# ELD Trip Planner

Full-stack app that plans FMCSA hours-of-service compliant truck trips. Enter your current location, pickup, dropoff, and hours already used in your 70hr/8day cycle — it maps the route with every required stop (fuel, 30-minute breaks, 10-hour rests, 34-hour restarts) and draws the driver's daily log sheets for each day of the trip.

- **Frontend:** Next.js (TypeScript, Tailwind CSS, Leaflet)
- **Backend:** Django + Django REST Framework
- **Routing/Geocoding:** OSRM + Nominatim (free, no API key)
- **E2E tests:** Playwright
